# PJfoot Feedback

This repository is used to collect feedback for the **PJfoot** app.

## How to Submit Feedback

Use the [Issues](../../issues) tab to submit your feedback:

- 🐛 **Bug report** – Something is not working as expected? [Open a bug report](../../issues/new?template=bug_report.md)
- 💡 **Feature request** – Have an idea for a new feature or improvement? [Open a feature request](../../issues/new?template=feature_request.md)

## Guidelines

- Search existing issues before opening a new one to avoid duplicates.
- Provide as much detail as possible to help us understand and reproduce the problem.
- Be respectful and constructive in your feedback.
