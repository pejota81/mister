# MISTER v0.26.1 - Admin Accounts Action Fix

> Released: 2026-05-17

Patch release to restore Admin Center account actions in production.

---

## Fixes

- **Account actions restored:** Fixed Admin Center account endpoints returning `HTTP 404` for account-type updates and account deletion.
- **Account type updates:** Changing account type now persists correctly instead of falling back to `basic`.
- **Account deletion:** Deleting accounts now works again through the Admin Center actions column.

## Technical

- Resolved Azure Functions route registration conflict in `api/src/functions/admin/accounts.js`.
- Consolidated route registrations for user account mutations into single handlers per route with multi-method support (`PATCH` + `DELETE`) for:
  - `admin/accounts/{userId}`
  - `admin-account/{userId}` (compat alias)
