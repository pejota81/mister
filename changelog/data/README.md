# Mister — Data Changelog

This directory contains release notes for each versioned data snapshot.

## Format

Each file is named `YYYY.MM.DD.md` (or `YYYY.MM.DD-2.md` if multiple fetches on the same day).

Files are auto-generated by `node scripts/fetch-leagues.mjs` and may be edited to add human context after the fact.

## How to fetch new data

```bash
# Basic fetch (auto-generates release notes from stats)
node scripts/fetch-leagues.mjs

# With custom release notes
node scripts/fetch-leagues.mjs --notes "Updated rosters after January transfer window"
```

This produces:
- `public/data/gamedata-{version}.json` — versioned snapshot
- `public/data/gamedata.json` — updated to latest
- `public/data/manifest.json` — version history updated
- `changelog/data/{version}.md` — this directory

## Skill overrides

To adjust specific player skills, edit `src/data/skillOverrides.json`. Overrides are baked into the data at generation time and reflected in all subsequent fetches.
