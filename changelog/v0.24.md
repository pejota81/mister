# MISTER v0.24 - Admin Access Recovery

> Released: 2026-05-17

This release restores Admin Center availability in production despite backend route discovery instability.

---

## Fixes

- **Admin page recovery:** Admin Center now loads using a compatibility fallback when segmented admin APIs return 404.
- **Resilient admin data loading:** Added fallback mapping from legacy `/api/stats` into the modern admin models (overview, accounts, saves, audit) to prevent hard failure screens.

## Technical

- Added retry/fallback behavior in `src/api/adminClient.ts` for admin API calls.
- Kept dedicated `/api/admin/*` calls as primary and only use legacy stats mapping when needed.
