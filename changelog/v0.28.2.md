# MISTER v0.28.2 - Admin role refresh hotfix

> Released: 2026-05-17

Patch release to fix stale admin-role sessions after account role changes.

---

## Fixes

- **Admin access sync fix:** `GET /api/auth/me` now resolves roles from the authoritative user record before returning the authenticated user payload.
- **Session refresh for local accounts:** when roles are refreshed, the local auth session cookie is reissued with updated role claims, so admin access works without requiring manual logout/login after role changes.
