# Mister v0.01 — League Rankings & New Manager Experience

> Released: 2026-05-02

---

## New Features

### IFFHS Global League Ranking
- All 9 divisions are now ranked according to the IFFHS Global Ranking
- Divisions are named by their global rank — **First Division** through **Ninth Division** — with the country name in parentheses
- The ranking order is reflected consistently across all screens: Main Menu, Dashboard, Table, Match Day, and Season End

### Global Promotion & Relegation Pyramid
- All 9 divisions now form a single worldwide promotion/relegation ladder
- **3 clubs promoted** from every division (except the First Division)
- **3 clubs relegated** from every division (except the Ninth Division)
- At season end, the bottom 3 clubs of each division swap with the top 3 of the division below, based on IFFHS rank order

### New Manager Start Experience
- New managers now start at the **bottom of the pyramid** (Ninth Division)
- The league selector has been removed from the new game setup form
- A motivational message explains the journey: start from the Ninth Division and climb to win the First Division
- Club list is automatically populated with all clubs from the lowest-ranked division

## UI Changes

### Login Screen
- "Continue with" label moved above the login buttons
- GitHub and Microsoft buttons are now displayed **side by side**

### Release Notes Link

## Bug Fixes

- Fixed `getPromotionRelegation` returning the entire table when `relegationSpots` or `promotionSpots` was 0 (caused by `slice(-0)` evaluating as `slice(0)`)
- Fixed `countryCode` field on `League` type being required but absent from generated data — marked as optional
