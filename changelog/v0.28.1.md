# MISTER v0.28.1 - Admin Center security + games terminology

> Released: 2026-05-17

Patch release focused on Admin Center hardening and terminology consistency.

---

## Improvements

- **Admin access locked down again:** `/admin` now requires authenticated users with `admin` role on both frontend gate and backend admin endpoints.
- **Game mode visibility in Admin Center:** admin data now includes mode usage details (`Pyramid` vs `Single League`) in metrics and account game details.
- **Terminology update:** Admin Center UI now uses **games** instead of **saves** in user-facing labels.
- **Account deletion integrity:** deleting an account now also removes all related games for that account, including fallback cases where only game records exist.
