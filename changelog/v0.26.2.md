# MISTER v0.26.2 - Admin Account Actions + Save Details

> Released: 2026-05-17

Patch release for Admin Center account-management reliability and account details visibility.

---

## Fixes

- **Account type updates:** fixed persistent failures where account type reverted to `basic` after selection.
- **Account deletion:** fixed Admin Center account deletion failures returning `HTTP 404`.
- **Compatibility hardening:** added static fallback mutation routes to tolerate environments with inconsistent route-parameter discovery.

## New

- **Account details saves panel:** account edit/details modal now shows the user's saves (save name, club, progress, last active), restoring previously available context.

## Technical

- Added backend static compatibility routes in `api/src/functions/admin/accounts.js`:
  - `PATCH /api/admin-account-update` (reads `userId` from body)
  - `DELETE /api/admin-account-delete?userId=...`
- Updated `src/api/adminClient.ts` fallback chain for account update/delete to call static compatibility paths when route-parameter paths return 404.
- Updated `src/components/screens/AdminScreen.tsx` to fetch and render save details for the selected account modal.
