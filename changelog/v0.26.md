# MISTER v0.26 - Admin Center Account Management

> Released: 2026-05-17

Focused Admin Center update for account lifecycle operations and US English naming consistency.

---

## Changes

- **Admin Center naming:** Standardized admin UI naming to **Center** (US English).
- **Accounts-focused admin flow:** Kept the stats cards and removed the button row below stats to focus the page on account management.
- **Accounts table redesign:** Updated columns to `E-mail/account`, `Name`, `Account type`, `Status`, `Created`, and `Last login`.
- **Row edit flow:** Clicking an account row now opens an edit screen/modal for account updates.
- **In-grid lifecycle controls:** Added direct grid actions to suspend/activate, delete accounts, and move users between account types (`basic`, `vip`, `admin`).

## Technical

- Added account deletion endpoint support in admin API:
  - `DELETE /api/admin/accounts/{userId}`
  - `DELETE /api/admin-account/{userId}` (compat alias)
- Account deletion removes both the user profile document and all saves tied to that user.
- Added `deleteAdminAccount()` client helper in `src/api/adminClient.ts`.
