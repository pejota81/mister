# Mister v0.20 — Auth Flow Simplification

> Released: 2026-05-16

This release simplifies the account access experience on the Main Menu so new users see a clearer, less overwhelming sign-in flow.

---

## UX Improvements

- Reworked Main Menu auth into explicit modes: `Login`, `Register`, `Forgot password`, and `Continue with`.
- Local account flows now show only local account fields/actions at once, instead of mixing all options together.
- Social providers (GitHub and Microsoft) now appear in a dedicated mode to reduce visual noise and decision overload.
- Added a shared auth-mode switch helper to keep transitions consistent and clear prior auth/recovery messages when changing flows.

---

## Operations

- Production auth configuration was remediated by setting `AUTH_SESSION_SECRET` in Azure Static Web App app settings, restoring local account registration/login session issuance.
