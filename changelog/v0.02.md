# Mister v0.02 — Squad Management & Player Depth

> Released: 2026-05-02

---

## New Features

### Fatigue System
- Players now have two new physical attributes: **Stamina** (fixed, 0–100) and **Fitness** (dynamic, 0–100)
- Playing in a match depletes fitness: GK −10, DEF −13, MID −15, FWD −17 per week
- Resting players recover fitness each week based on their stamina (`stamina ÷ 5` per week)
- Fitness directly affects match performance: `effective skill = skill × (0.6 + 0.4 × fitness/100)`
- A fully exhausted player (fitness 20) performs at only 68% of their rated skill
- Fitness is fully restored during the off-season
- **Squad rotation is now a real strategic decision**

### Negotiable / Transfer Listing
- Managers can now mark players as **negotiable** (listed for sale) directly from the Squad screen
- Toggle with the ☆/★ button — highlighted amber when listed
- The Transfer Market now only shows **negotiable players** and free agents
- Renewing a contract automatically removes the negotiable flag

### Contract Renewal
- New **Renew** button on every player row in the Squad screen
- Contract length is recalculated based on player age using the same rules as initial contracts
- Salary is updated to current market rate (`calcSalary(skill)`)
- A **signing bonus** (8× weekly salary) is charged upfront
- Button is highlighted when contract ≤ 1 season remaining and disabled if the club cannot afford the bonus
- Confirmation shows: new contract length, new weekly salary, and signing bonus cost

### Sortable Squad Table
- Every column in the Squad screen is now **clickable to sort**
- Clicking the same column toggles between ascending ▲ and descending ▼
- **Team A and Team B tables sort independently**
- Default sort: Position → Skill

---

## Contract Model Overhaul

- Contracts are now measured in **seasons** (not weeks)
- Contract lengths are age-based:
  - **U22:** 4–5 seasons
  - **22–32:** 1 season (20%), 2 seasons (60%), 3 seasons (20%)
  - **33–34:** 1–2 seasons
  - **35+:** 1 season
- Contract column displays as `2s`, shown in red when ≤ 1 season remaining
- All existing player data migrated to the new model

---

## Season Model

- Game now starts on **Season 2026** (calendar year)
- **Mandatory retirement** at season start: all players aged > 38 are retired
- Each mandatory retiree is replaced by a new **18-year-old player** with a 5-season contract
- This runs at the start of every season, including the very first

---

## Player Value & Salary

- `calcSalary()` and `calcMarketValue()` are now a **single source of truth** in `src/engine/playerValue.ts`
- Market value is recalculated every season end as players age (previously frozen at generation time)
- Youth and replacement players now use the same age-tiered formula as generated data
- `calcStamina()` added to the shared utility — generates consistent stamina ratings from player ID + age

---

## Squad Screen Improvements

- **Nationality column** now shows flag emoji + 3-letter code (e.g. 🇪🇸 ESP) instead of full country name
- **Skill bar removed** — replaced with a colored number (green ≥80, amber ≥70, dim below)
- **Fitness column** added — shows current fitness % in green/amber/red
- **Release confirmation** now shows full financial breakdown:
  - A. Severance pay (immediate cost)
  - B. Total wages saved (remaining contract value)
  - Net balance (B − A)

## Transfer Market

- Only players explicitly **listed as negotiable** appear in the market (plus free agents)
- Previously all non-managed-club players were shown
