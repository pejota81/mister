# PJfoot v0.00 — Initial Release

> Retro-styled football manager simulator inspired by Elifoot 98.

---

## Core Game Concept

- Single-player football club management in a league competition
- Text-based, terminal-aesthetic UI (DOS/Elifoot 98 inspired)
- Deterministic match engine with seeded randomness (replay consistency)
- Multi-season progression with week-by-week advancement

---

## Leagues & Competitions

- Multiple international leagues with real football data:
  - England (Premier League, Championship)
  - Spain (La Liga)
  - Germany (Bundesliga)
  - Italy (Serie A)
  - France (Ligue 1)
- Multiple divisions per league with promotion and relegation
- Double round-robin league fixtures (home & away) — up to 38 game weeks
- **Cup Competition:** Single-elimination knockout tournament
  - Rounds from Round of 64 to Final
  - Extra time (30 min) and penalty shootouts if drawn
  - Goalkeeper skill influences penalty save probability

---

## Club Management

- Select and manage any club from available leagues and divisions
- Club attributes: stadium capacity, ticket price, squad size, cash balance
- Stadium expansion (increases capacity by 25%, scales in cost)
- Ticket price management with revenue impact

---

## Match Simulation

- Strength calculation based on:
  - Weighted average skill of starting XI (position-weighted: GK 1.2×, FWD 1.1×)
  - Formation multipliers (attack/defense balance)
  - Mentality modifiers (Defensive / Balanced / Attacking)
  - Home advantage (5% boost)
- ~6 attack attempts per team per 90 minutes
- Scoring probability based on attacker vs defender strength ratio
- Random scorer selection from forwards/midfielders

---

## Squad & Tactics

- **Formations (6 available):** 4-4-2, 4-3-3, 3-5-2, 5-3-2, 4-5-1, 3-4-3
- **Mentality:** Defensive, Balanced, Attacking
- Starting XI selection (11 players) with position validation
- Substitutes bench (up to 5 players)
- Team B reserve squad (move players in/out of main squad)
- Squad size limit: 30 players total
- Default lineups auto-generated if not manually set

---

## Player Attributes & Progression

- **Attributes per player:** Name, nationality, position, age, current skill (0–100), peak skill, weekly salary, market value, contract weeks remaining
- **Skill progression by age:**
  - Growth from age 17 (55% of peak) to peak age (26–29)
  - Decline after peak: 1.2%/year until 32, then 2.5%/year
  - Minimum skill floor: 35
- **Retirement probability by age:** 5% at 33, rising to 90% at 38+
- **Youth emergence:** 1–3 new youth players (ages 17–19) generated per retiree
- Real player database generated from actual football data

---

## Transfer Market

- Search players by name with position filter (All / GK / DEF / MID / FWD)
- Free agent detection and acquisition
- Minimum transfer fee: 90% of market value
- Player release with severance pay (contract weeks left / 4 × weekly salary)
- Transfer history log (week, player, clubs, fee)
- AI clubs automatically manage rosters between seasons (release elderly players, sign free agents)

---

## Financial System

- **Income:** Matchday ticket revenue (dynamic attendance based on team quality, ±5% noise, 55–85% capacity base)
- **Expenses:** Weekly player salaries + loan repayments
- **Season-end prize money:** Champion, runner-up, and third-place bonuses
- **Loan options:**
  - $500k over 12 weeks
  - $1M over 20 weeks
  - $2M over 40 weeks
  - $5M over 52 weeks
- Stackable loans with weekly repayment tracking

---

## Season Flow

1. Select formation, mentality, and starting XI
2. Advance week — all fixtures (league + cup) auto-simulated
3. View match result, goal scorers, and strength ratings
4. Review full division table and other results
5. Manage transfers, finances, and squad
6. Repeat until season end (week 38+)
7. Season-end screen shows final standings, promotions/relegations, prize money, aging events
8. Season rolls over: new fixtures, updated squads, year increments

## Localization

- Languages: **English**, **Portuguese (pt-BR)**, **Spanish**
- Language switcher (persisted via localStorage)
- Dynamic switching without page reload
- Locale-specific number and currency formatting

---

## Save System & Authentication

- Cloud-based saves via Azure Functions + Cosmos DB
- OAuth 2.0 authentication (GitHub, Microsoft)
- Guest mode (no save persistence)
- Auto-save after game mutations with status indicator (saving → saved)
- Compressed save format (delta storage — only changes from base data)
- Load and delete saves from the Load Game screen

## Visual & Technical

- Retro terminal aesthetic: dark background, green (#33cc33) / amber (#ffaa00) color scheme, Courier New monospace font, scanline overlay
- Skill bars with color coding (green >80, amber 70–79, red <70)
- League table zones highlighted (promotion green, relegation red)
- Deterministic LCG (Linear Congruential Generator) for seeded match randomness
- React 19 + TypeScript frontend, Vite build tool
- Azure Static Web Apps deployment
- i18next for localization
