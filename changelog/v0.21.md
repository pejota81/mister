# Mister v0.21 - Admin Center Modernization

> Released: 2026-05-16

This release modernizes the Admin Center to support game account operations and cleaner admin workflows as first-class features.

---

## Admin Experience

- Replaced the single admin view with tabbed sections: `Overview`, `Accounts`, `Saves`, `Risk / Recovery`, and `Audit Log`.
- Added account lifecycle actions directly in admin: suspend/activate accounts and rotate recovery codes for local users.
- Kept save moderation flows while separating account management from game-save operations.

---

## Backend API

- Introduced dedicated admin endpoints:
  - `GET /api/admin/metrics`
  - `GET /api/admin/accounts`
  - `PATCH /api/admin/accounts/{userId}`
  - `POST /api/admin/accounts/{userId}/recovery-code`
  - `GET /api/admin/saves`
  - `GET /api/admin/audit`
- Added shared admin backend helpers for Cosmos access/retry and account normalization.
- Updated route registration to load the new admin modules explicitly.

---

## Fixes Included

- Fixed profile date-of-birth persistence so saved DOB no longer resets unexpectedly.
- Fixed Admin Center user listing to include newly created accounts even before they create any saves.
