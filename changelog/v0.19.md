# Mister v0.19 — Accounts, Profile Settings & Password Recovery

> Released: 2026-05-16

This release introduces first-party user accounts, persistent profile settings, and a full password recovery flow.

---

## New

- Added native account registration and login with email/password.
- Added user roles (`basic`, `vip`, `admin`) in account/session data.
- Added persistent profile endpoints and UI for:
  - display name
  - manager defaults (name, nationality, date of birth)
  - language preference
  - theme preference
- Added password recovery:
  - reset via recovery code
  - recovery code generation from account settings
  - recovery code rotation after successful reset

---

## UX Improvements

- Main Menu now centralizes account maintenance and preferences.
- New Game setup pre-fills manager fields from saved profile defaults.
- Account settings controls are localized for EN/ES/PT-BR.
- Account settings layout improved for mobile screens.
